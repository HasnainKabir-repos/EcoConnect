## Getting Started

To set up the project locally you need to follow the following steps

### Installation

Install all dependencies

- npm

  ```sh
  npm install
  ```

  - npm

  ```sh
  cd backend
  npm install
  ```

- npm
  ```sh
  cd ../frontend
  npm install
  ```

### Tests

- npm
  ```sh
  npm run test
  ```

### Run

- npm
  ```sh
  npm run dev
  ```
